# 작업 규칙

## Claude에게 바라는 것
1. **완성된 파일 전체**를 줄 것 — 코드 조각이 아니라 붙여넣기 가능한 파일
2. 설명은 짧게, 한국어로
3. 단계마다 "지금 실행해보고 오세요" 한 마디로 마무리
4. 에러가 붙여넣어지면 즉시 분석·수정
5. 기존 기능을 건드리지 않고 추가만 할 것
6. 패키지 설치 명령어는 항상 같이 알려줄 것

## 절대 하지 말 것
- `config.json` / `brain.json` 내용을 코드에 하드코딩하지 말 것
- API 키를 코드 안에 넣지 말 것
- 기존 동작하는 코드를 허락 없이 리팩터링하지 말 것
- 한 번에 너무 많은 것을 바꾸지 말 것 (단계별로)
