# 나에 대해

## 역할
- 비개발자 — 코딩 없이 Claude와 대화하며 앱을 만드는 사람 (바이브 코딩)
- 개인 지식 관리 도구를 직접 설계·기획하는 사람

## 현재 작업 중인 프로젝트
- **Second Brain Desktop** (`C:\Users\gustj\Brain\`)
  - Python + Tkinter 기반 개인 노트·지식 관리 데스크탑 앱
  - Gemini API 연동 (aistudio.google.com 무료 티어)
  - GitHub: https://github.com/gustj369/second-brain-desktop

## 개발 환경
- OS: Windows
- Python 3.9+, Tkinter (기본 포함)
- 에디터: Claude Code (AI 페어 프로그래밍)
- 패키지: requirements.txt 참고
- 데이터: `brain.json` (로컬), `config.json` (API 키)

## 목표
- 내 사고방식에 맞춘 노트 도구를 직접 만들어 사용하기
- "비개발자도 AI와 함께라면 만들 수 있다"는 것을 증명하기
- 기능이 쌓일수록 내 노트가 더 유용해지는 경험 만들기
